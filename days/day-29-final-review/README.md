## Common Exam Traps to Avoid

### Watch Out For

**1. "NOT" Questions**
- "Which is NOT true..."
- "Which does NOT apply..."
- Read carefully!

**2. Absolute Words**
- "always", "never", "must", "cannot"
- Usually indicate wrong answers

**3. "All of the Above"**
- Verify each option is correct
- If one is wrong, eliminate this choice

**4. Similar Options**
- Look for subtle differences
- Consider edge cases and limitations

**5. Scenario Questions**
- Identify the key requirement
- Consider cost, performance, security
- Choose "most appropriate" solution
